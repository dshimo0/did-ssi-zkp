# Contribuindo

Obrigado por ajudar a expandir este projeto. A regra principal e simples: toda informacao factual importante precisa ser rastreavel.

## Politica de fontes

Ao adicionar ou alterar conteudo:

- Prefira fontes primarias: W3C, OpenID Foundation, IETF, Governo Federal, Planalto, ITI, Hyperledger, DIF ou documentacao oficial do projeto citado.
- Inclua a fonte em [docs/sources.md](docs/sources.md) quando ela ainda nao existir.
- Use a fonte no texto proximo da afirmacao, nao apenas no fim do arquivo.
- Marque como "hipotese", "inferencia" ou "recomendacao" quando nao for um fato diretamente afirmado pela fonte.
- Nao use dados pessoais reais em exemplos.

## Exemplos

Arquivos em `examples/` devem declarar se sao:

- sinteticos;
- adaptados de especificacoes;
- derivados de documentacao oficial;
- ou dados reais anonimizados.

Por padrao, use apenas dados sinteticos.

## Criterios de aceitacao

Uma contribuicao esta pronta quando:

- o texto aponta para a origem dos dados;
- exemplos nao expõem dados pessoais reais;
- links relevantes foram revisados;
- termos tecnicos estao alinhados com os padroes citados.
